Tweemoji Support

Published on March 2019 as syntax.

Emoji can be enabled in a Hugo project in a number of ways. In this theme, you can use Twemoji kit simply writing emoji and then adding this param in the config.toml file.

twemoji = false

More info in the theme wiki on Github.
